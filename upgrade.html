Major Release

What's New in v11.1

Version 11.1 is a complete architecture overhaul of the ALKNZ Intelligence Chrome Extension. From per-user authentication and smart capture modes to AI-powered recommendations and team conflict detection, this release transforms the extension from a single-user scraper into a collaborative research platform.

New

Per-User Authentication

v8 used a single shared API key with a manually typed "Captured By" field. v11.1 introduces real platform authentication so every team member logs in with their own credentials and all captures are attributed individually.

v8.0 (Old)
  • Shared master API key for all users
  • Manual "Captured By" text input
  • No identity verification
  • No per-user tracking on backend
v11.1 (New)
  • Email + password login in Options page
  • POST /api/auth/login returns Bearer token
  • Token stored securely in chrome.storage.sync
  • All API calls include Authorization: Bearer
  • Captured-by auto-set from authenticated session
New

Smart Capture Modes

The extension now auto-detects the type of page you're viewing and switches to the appropriate capture mode. You can also toggle modes manually via the mode banner at the top of the Research tab.

Person Mode
Capture individual investor profiles with contact info, enrichment, and investment focus fields.
Company Mode
Capture firm profiles with org type, HQ location, investment sectors, stages, and check size range.
Bulk / Team Mode
Select multiple team members from a company page and enrich them sequentially in a batch queue.
Auto-detection: The content script identifies the page type (person_profile, company_profile, team_page) using a 3-layer extraction strategy: JSON-LD structured data, LinkedIn-specific DOM selectors, and generic meta tags.
Redesigned

Inline Enrichment Panel

In v8, AI enrichment lived in a separate "Deep AI" tab. In v11.1, enrichment is embedded directly in the Person capture form as a collapsible section. Results appear as individual field rows with Apply and Dismiss buttons for full user control.

v8.0 Enrichment
  • Separate "Deep AI" tab
  • Terminal-style output (raw text)
  • Bulk merge button (all or nothing)
  • No per-field control
v11.1 Enrichment
  • Inline panel inside capture form
  • Clean card UI with structured field rows
  • Per-field Apply / Dismiss buttons
  • Confidence indicators
  • Enrichment status bar with progress
New

Team Conflict Detection

When two team members capture the same contact, v11.1 shows a warning before upload. This prevents duplicate outreach and gives the admin platform-side visibility.

Example Warning
"Khaled already captured this person (Jan 15). Uploading will add your version."
Upload still proceeds — no blocking. The admin dashboard flags contacts with 2+ team members for CEO review.
Powered by GET /api/v1/people/lookup — the extension checks for existing records before every upload.
New

Post-Upload Recommendations

After a successful upload, the extension now suggests similar investors to research next and provides approach suggestions for the captured profile.

Research Next
After uploading, a "Research Next" card appears with 3-5 similar profiles based on matching investment sectors, stages, and geography. Each suggestion links directly to the investor's LinkedIn profile.
Scoring: Sectors (0.4) + Stage (0.3) + Geography (0.2) + Firm type (0.1)
Suggested Approach
A "Next Steps" card provides tactical outreach advice: intro path (warm vs. cold), key talking points based on shared sectors, portfolio reference points, and suggested timing.
Rule-based engine. No ML required for initial release.
New

Company Capture Mode

v8 only captured person profiles. v11.1 adds a dedicated company form with its own fields and upload endpoint.

Company Fields
  • Organization name & type
  • Website & LinkedIn URL
  • Up to 5 emails, 3 phone numbers
  • HQ city & country
  • Description & notes
Investment Profile
  • Investment sectors
  • Investment stages
  • Check size range (min / max)
  • Geographic focus
Company upload: POST /api/v1/companies (fallback: /api/organizations)
Redesigned

Tab Reorganization & Progressive UI

The extension UI has been streamlined from 4 tabs down to 3. The Deep AI tab has been removed and its functionality moved inline. Forms now use progressive disclosure with collapsible sections.

Research
Main capture form with mode banner, progressive fields, inline enrichment, and upload.
Batch
Scraper results with expand/collapse rows. Smart URL pre-fill from detected company pages.
Saved
Drafts, address book, and search history combined in one unified view.
Progressive Form Sections
Contact & Relationship — collapsible, starts collapsed. Reduces visual noise when you only need to capture basic info.
Investment Focus — collapsible, starts collapsed. Expand only when capturing investment-specific data like sectors, stages, and geo focus.
Upgraded

3-Layer Data Extraction

The content script has been rewritten with a smarter, more reliable extraction pipeline.

1
JSON-LD Structured Data
Highest priority. Extracts rich schema.org data embedded in the page.
2
LinkedIn DOM Selectors
Platform-specific selectors for LinkedIn profile and company pages.
3
Generic Meta Tags
Fallback. Reads OpenGraph, Twitter cards, and standard meta description.
Updated

New & Updated API Endpoints

v11.1 introduces new endpoints and uses primary + fallback patterns for backward compatibility.

Endpoint Fallback Purpose
POST /api/auth/login — User authentication
GET /api/auth/me — Token validation
POST /api/v1/people /api/capture Person upload
POST /api/v1/companies /api/organizations Company upload
POST /api/v1/enrichment /api/deep-search AI enrichment
POST /api/v1/extract /api/extract AI text extraction
POST /api/v1/scrape /api/batch-scrape Batch scraping
GET /api/v1/people/lookup — Conflict check + dedup
GET /api/v1/people/{id}/recommendations — Similar investor suggestions
GET /api/v1/people/{id}/next-steps — Approach suggestions
All new v1 endpoints try the primary route first. If it returns 404, the extension automatically retries on the fallback route for backward compatibility with older backends.
How To

How to Upgrade

Upgrading from v8 to v11.1 takes about 2 minutes. Your settings (server URL, API key) are preserved.

1
Download v11.1
Get the latest ZIP from the Installation page.
2
Remove the old extension
Go to chrome://extensions, find ALKNZ Intelligence v8, and click Remove.
3
Extract and load unpacked
Extract the v11.1 ZIP, then click Load unpacked and select the folder containing manifest.json.
4
Log in with your platform account
Open Options, enter your email and password, and click Log In. Your server URL and API key should still be set from v8.
You're done
Open any LinkedIn profile and the extension should auto-detect the page type and switch to the right mode.
