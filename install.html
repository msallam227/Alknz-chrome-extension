Install & Configure ALKNZ Intelligence

This extension is deployed as a private internal build. You will install it using Chrome Developer Mode, then connect it to the ALKNZ backend using a server URL + master API key.

Download Package
alknz-extension-v8.0.0.zip
Load as “unpacked” after extracting
Download
Step 1

Enable Developer Mode

Open Chrome Extensions, then enable Developer Mode.

chrome://extensions
Toggle Developer mode (top-right).
Step 2

Load Unpacked Extension

Extract the ZIP file, then click Load unpacked and select the extracted folder.

✅ Correct
  • Select the folder that contains manifest.json
  • Extension appears in the list immediately
  • Pin it to the Chrome toolbar
❌ Common mistake
  • Selecting the ZIP without extracting
  • Selecting the parent folder above manifest.json
  • Using Safari or non-Chrome browsers
Step 3

Configure Backend Connection

Open the extension, go to Settings (gear icon), then paste: Backend Server URL + Master API Key.

Backend Server URL
https://YOUR-SERVER-URL-HERE
Must be reachable from your network. If this is Replit or a private host, ensure it’s online.
Master API Key
REPLACE_WITH_MASTER_KEY
Internal secret. Do not commit into GitHub. Store in a password manager.
Security Note

If this documentation site will ever be hosted publicly, remove all keys and replace with placeholders. Keys should be provided to team members via secure channels only.

Step 4

Verify It’s Working

Use these checks to confirm your extension is connected and ready.

Checklist
  • Extension icon visible in toolbar (pin if needed)
  • Settings saved (URL + Key)
  • Status shows Connected (or no auth error)
  • Try a known investor page and click Capture
What you should see
  • A detected entity name
  • Scraped fields populated (if available)
  • Deep AI enrichment (if enabled)
  • Upload confirmation / success state

Troubleshooting

If something doesn’t work, it’s usually one of these:

“Load unpacked” is greyed out
Make sure you enabled Developer mode at the top right of chrome://extensions.
Extension loads but doesn’t capture anything
Most common causes:
  • You’re on a page type the scraper doesn’t support yet
  • Permissions were denied (refresh the page after installing)
  • Backend URL is wrong/offline
401 / Unauthorized
Your Master API Key is missing, incorrect, or the backend is validating a different key. Re-enter the key in Settings and retry.
CORS / Network error
This usually means the backend is not allowing requests from the extension origin. Confirm the backend is configured for the extension’s requests and that the server is reachable.
Extension updated — now things look “stuck”
Go to chrome://extensions → click the refresh icon on the extension card. Then refresh your browser tab.
